# Tara for Boulder

Made with [Doks Starter theme](https://github.com/h-enk/doks)

## Requirements

- [Git](https://git-scm.com/) — latest source release
- [Node.js](https://nodejs.org/) — latest LTS version or newer

## Get started

### 1. Install dependencies

```bash
npm install
```

### 3. Start development server

```bash
npm run start
```
